Chapter 4 Terms
Term Definition
Network Classifications
  • LAN
  • MAN
  • WAN
  • Local Area Network

  • Metropolitan Area Network

  • Wide Area Network
Network models
  • Bus Network
  • Star Network
  • LAN topology in which each node, workstation, or other device is connected to a main cable or link called a bus

  • LAN topology in which all nodes, PCs, workstations, or other devices are directly connected to a common central computer that is often referred to as a hub.
Protocols
  • CSMA/CD Ethernet
  • CSMA/CA Wifi
  • Carrier-sense multiple access with collision detection. method used for early local area networking

  • Carrier-sense multiple access with collision avoidance. minimize collision occurring when two or more stations send their signals over a data link layer.
Combining Networks
  • Repeater
  • Bridge
  • Switch
  • Router
  • Gateway
  • device that passes signals back and forth between the two original buses

  • looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side

  • a bridge with multiple connections, allowing it to connect several buses rather than just two

  • purpose-built computers used for forwarding data

  • The point at which one network is linked to an internet is often called a gateway because it serves as a passageway between the network and the outside world
Client/sever model describes the communication between two computing entities over a network.
  • P2P: peer-to-peer
  • distributed system
  • cloud computing
  • the peer-to-peer model involves processes that provide service to and receive service from each other

  • consist of software units that execute as processes on different computers.

  • huge pools of shared computers on the network can be allocated for use by clients as needed, is the latest trend in distributed systems
Internet Addressing
  • Tier 1
  • Tier 2
  • Tier 3
  • ISPs that consist of very high-speed, high-capacity, international WANs. These networks are thought of as the backbone of the Internet.

  • ISPs that tend to be more regional in scope and less potent in their capabilities.

  • intermediary called an access, when tier 1 and tier 2 fail.
IP address
  • ICANN
  • DNS
  • TLD: Top-level domain
  • Country-code TDLs
  • Internet Corporation for Assigned Names and Numbers (ICANN), which is a nonprofit corporation established to coordinate the Internet’s operation

  • domain name system, translate human readable domain names to readable IP addresses

  • highest level in the domain name system

  • TLD used or reserved for a country
Internet protocols
  • FTP
  • HTTP
  • SMTP
  • File Transfer Protocol (FTP), or an application for accessing another computer from a great distance

  • Secure shell protocol, cryptographic network protocol for operating network services securely over an unsecured network

  • Simple Mail Transfer Protocol, defines a way that two computers on the network may interact when transmitting an email message from one host to the other.
World Wide Web
  • URL
  • Client-side activities
  • server-side activities
  • Uniform Resource Locator, In order to locate and retrieve documents on the Web, each document is given a unique address

  • Everything in a web application that is displayed or takes place on the client side

  • runs on the server instead of the client
Internet four-layer software
  • application layer
  • transport layer
  • network layer
  • link layer
  • Prepares message and provides destination address, also receives message

  • Chops message into packets, and collects packets and reassembles message.

  • Assigns intermediate address to each packet, and detects that packet has reached its final destination

  • transfers and receives packets
Internet is:
  • packet-switching network
  • fault tolerant
  • redundant
  • Type of computer communications network that groups and sends data in the form of small packets.

  • enables system to keep functioning in the event of a failure

  • more than what is necessary
Cybersecurity
  • malware
  • virus
  • worm
  • Trojan horse
  • spyware
  • phishing
  • DDoS attack
  • Many attacks incorporate the use of malicious software (collectively called malware). Such software might be transferred to, and executed on, the computer itself, or it might attack the computer from a distance

  • software that infects a computer by inserting itself into programs that already reside in the machine. Then, when the “host” program is executed, the virus is also executed. W

  • autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.

  • a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim.

  • software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack

  • a technique of obtaining information explicitly by simply asking for it, often carried out via email.

  • a distributed denial of service; the work of generating the attacks is distributed over many hosts
Protection
  • firewall
  • spam filter
  • proxy server
  • antivirus software
  • cryptography
  • public key encryption
  • A primary prevention technique to filter traffic passing through a point in the network

  • firewalls designed to block unwanted email

  • a software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.

  • is used to detect and remove the presence of known viruses and other infections.

  • the study of sending and receiving messages securely in the presence of adversaries

  • techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
