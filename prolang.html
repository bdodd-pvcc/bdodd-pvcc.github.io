Chapter 6 Terminology

Term Definition
Machine language Lowest-level programming language.
Debugging The process of finding and removing errors (also called bugs) in a computer program or a piece of electronic hardware that can cause it to behave unexpectedly or crash.
Assembler Type of computer program that converts instructions written in assembly language into machine code, which can be executed by a computer.
Assembly language Language that equates to machine code but is more readable and uses mnemonics. The statements are made up of opcodes and operands, which are directly translated into machine code. Each CPU has its own assembly language.
Translator A program that translates from one programming language into another.
Compiler In computing, a computer program that translates computer code written in one programming language (the source language) into another language (the target language).
Interpreter A program that can analyze and execute a program line by line.
Cross-platform software Creating software that will be implemented and run on multiple platforms.
Natural language A language that has developed naturally in use (as contrasted with an artificial language or computer code).
Formal language Used among others as the basis for defining the grammar of programming languages and formalized versions of subsets of natural languages in which the words of the language represent concepts that are associated with particular meanings or semantics.
Machine Independence Referring to programs and procedures which function in essentially the same manner regardless of the machine on which they are carried out.
High-Level Programming Language A programming language that resembles natural language or mathematical notation and enables a programmer to write programs that are more or less independent of a particular type of computer.
Object-Oriented Programming (OOP) A software programming model constructed around objects. This model compartmentalizes data into objects (data fields) and describes object contents and behavior through the declaration of classes (methods).
Scripting Language A programming language that uses scripts to automate tasks, create media, or interact with other languages
